# qAttitude

qAttitude © 2026 by Andrea Bistacchi, released under GNU AGPLv3 license.

QGis plugin for on the fly orientation analysis of geological data, with stereoplots and orientation statistics.

To install, first install the PackageInstallerQgis plugin by BRG, available in QGis under Plugins > Manage and Install Plugins. This is used to manage requirede libraries for this and other plugins. Then download and place the [qAttitude folder](https://github.com/gecos-lab/qAttitude/qAttitude) in your QGis plugins folder, which under Windows is C:\Users\<your user name>\AppData\Roaming\QGIS\QGIS3\profiles\default\python\plugins. Then you will find the qAttitude plugin under Plugins > Manage and Install Plugins and you will be able to activate it.
